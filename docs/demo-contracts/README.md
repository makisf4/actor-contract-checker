## Demo Contracts (Regression Testing)

### Σκοπός
Τα παρακάτω κείμενα είναι 100% φανταστικά και χρησιμοποιούνται αποκλειστικά για δοκιμές παλινδρόμησης (regression testing) της εφαρμογής.

### Πώς να τα χρησιμοποιήσεις
1) Άνοιξε ένα από τα demo αρχεία.
2) Αντέγραψε όλο το κείμενο.
3) Επικόλλησε το στο πεδίο εισαγωγής της εφαρμογής.
4) Προχώρησε στην ανάλυση και έλεγξε τα αποτελέσματα.

### Τι να περιμένεις (υψηλού επιπέδου)
- **MOVIE_demo.md:** Περιλαμβάνει αμοιβή/πληρωμή, δικαιώματα εικόνας/φωνής, περιορισμένη αποκλειστικότητα, credits και αναφορά σε ψηφιακή χρήση. Λείπουν συγκεκριμένοι όροι (π.χ. territory/λύση).
- **SERIES_demo.md:** Περιλαμβάνει αριθμό επεισοδίων, ισχυρή αποκλειστικότητα, όρους λύσης και επαναχρησιμοποίηση υλικού. Δεν υπάρχει ρητή αναφορά σε AI.
- **TVC_demo.md:** Περιλαμβάνει διάρκεια χρήσης, μέσα προβολής, περιοχή, cutdowns/edits και buyout. Απουσιάζει σαφής τιμολόγηση ανανέωσης.

### Σημείωση
Τα κείμενα είναι φανταστικά και δεν αποτελούν νομική συμβουλή.
